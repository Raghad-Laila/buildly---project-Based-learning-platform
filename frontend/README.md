# نظام إدارة المسارات التعليمية - Frontend

هذا هو مشروع Frontend لنظام إدارة المسارات التعليمية مبني باستخدام React.

## المتطلبات

- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn

## التثبيت

1. قم بتثبيت الحزم المطلوبة:
```bash
npm install
```

## التشغيل

لتشغيل المشروع في وضع التطوير:

```bash
npm run dev
```

سيتم فتح التطبيق على `http://localhost:3000`

## البناء للإنتاج

لإنشاء نسخة للإنتاج:

```bash
npm run build
```

## البنية

```
frontend/
├── src/
│   ├── components/      # المكونات المشتركة
│   ├── contexts/        # Context API (المصادقة)
│   ├── pages/           # صفحات التطبيق
│   ├── services/        # خدمات API
│   ├── App.jsx          # المكون الرئيسي
│   ├── main.jsx         # نقطة الدخول
│   └── index.css        # الأنماط العامة
├── index.html
├── package.json
└── vite.config.js
```

## الميزات

- تسجيل الدخول والتسجيل (متعلم/مشرف)
- لوحة تحكم للمتعلمين والمشرفين
- إدارة المسارات التعليمية (عرض، إنشاء، تعديل، حذف)
- إدارة المشاريع التعليمية (عرض، إنشاء، تعديل، حذف)
- انضمام المتعلمين للمسارات
- الملف الشخصي
- تصميم احترافي متجاوب

## الاتصال بالباك إند

تأكد من أن الباك إند يعمل على `http://localhost:8000`

يمكنك تعديل عنوان الباك إند في `src/services/api.js`
